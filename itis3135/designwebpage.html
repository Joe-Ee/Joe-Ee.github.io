Design Proposal

  1. Overview

    1. Application and its Purpose

      This is an application used to teach javascript and html to beginners. The intended users are people who want to have a quick guide to basic javascript functions and how they help with coding web pages or beginners who want to learn javascript. The website will have multiple pages. There will be a page with beginner concepts like variable declaration, loops, and conditionals. Then there will be a page on the history of javascript and HTML and then a page on applying javascript functions in HTML with buttons and forms.

    2. Intended Users

      This website is meant for beginners

    3. Overview of the Content

      This website will have a variety of content. One page will have an overview of html and javascript. Another will have a html based page which goes over how html works, and then talks about commonly used tags and page formatting. There will be 3 javascript pages. One goes over simple things in javascript like declaring variables and basic programming fundamentals. The next one, will Go over decisino making and looping in javascript. The last page will be a page that shows the user ways to connect html buttons with javascript functions.

  5. Page design
    • Home
      1. This page will give an overview of HTML and Javascript with history and talk about their importance.
      2. This will be a page for the users of the website.
      3. The content will be paragraphs about javascript and HTML history and importance. There will be a button that switches between each language. There will be images to supplement the information.
      4. No user information will be entered into this page
      5. One button will be on this page to switch the language that the content is on.
    • Basics of Javascript
      1. Show the user the very basics of javascript
      2. This will be a page for the users of the website.
      3. The content will show users basic javascript syntax. It will show variable declaration, math, and putting text on a screen
      4. No user information will be entered into this page
      5. There will be a button that shows example programs with javascript. So the user will click the buttons and it will show the code and the code will run on the user's screen so they can see it. Ex: user presses a button and alert pops up. The website will show the code for the alert.
    • Loops and Conditionals
      1. Show the user javascript loop and conditional statement syntax.
      2. This will be a page for the users of the website.
      3. The content will be syntax of javascript loops and conditionals.
      4. Users will type in input to interact with page programs
      5. There will be code that shows users how conditionals and loops work. Similar to the last page, the user will press a button and then code will run. Also, there will be areas where users put in inputs and it shows how loops and conditionals.
    • HTML Syntax
      1. Show the user HTML syntax and page layout
      2. This will be a page for the users of the website.
      3. The content will be HTML syntax and how to properly structure a page for best user experience. Also some very basic CSS
      4. No user input

    • Javascript on HTML Pages
      1. Show the user how to make javascript interact with an HTML page
      2. This will be a page for the users of the website.
