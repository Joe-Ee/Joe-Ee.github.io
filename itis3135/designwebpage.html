Design Proposal

  1. Overview

    1. Application and its Purpose

      This is an application used to teach javascript and html to beginners. The intended users are people who want to have a quick guide to basic javascript functions and how they help with coding web pages or beginners who want to learn javascript. The website will have multiple pages. There will be a page with beginner concepts like variable declaration, loops, and conditionals. Then there will be a page on the history of javascript and HTML and then a page on applying javascript functions in HTML with buttons and forms.

    2. Intended Users

      This website is meant for beginners

    3. Overview of the Content

      This website will have a variety of content. One page will have an overview of html and javascript. Another will have a html based page which goes over how html works, and then talks about commonly used tags and page formatting. There will be 3 javascript pages. One goes over simple things in javascript like declaring variables and basic programming fundamentals. The next one, will Go over decisino making and looping in javascript. The last page will be a page that shows the user ways to connect html buttons with javascript functions.
