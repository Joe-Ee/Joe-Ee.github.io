Including Javascript Code in HTML

An important aspect of javascript is that it is interpreted in the browser. In order to include any javascript program into html you must have a pair of <script> tags. These are important because it allows for the recognition of javascript code. Within the tag you can either link to a file or just put the code between the open and closing tags. Javascript files

Using the innerhtml function in javascript

An easy way to change an html element's text is using the inner html function in javascript. This requires giving an element an id in the html file and then using that ID in the linked javascript to get and set the innerhtml of that element. the syntax looks like: document.getElementById("name").innerHTML.
