Loops and Conditionals in Javascript

Loops are easy ways to repeat part of a program multiple times in a row. For example if you wanted to print out something x times you can write a loop to do it. Conditionals are used in ways to make decisions. For example, if a statement is true the program will do one thing but if it is not the program will do something else. This could be used for error checking and other things.

If statement

This is a statement that checks if a variable meets a condition and then runs a certain piece of code based on the condition
