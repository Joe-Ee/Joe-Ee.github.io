Basics of Javascript

Variables

Javascript is a weakly typed langauge which means that you do not have to declare a datatype. To declare a variable in javascript you use the let or var keyword. to declare a constant you use the const keyword. Variables hold values and allow them to change. Constants hold values that do not change throughout the program. An example of a variable declaration is let variable = 1;. let is nicer to use because you cannot redeclare the variable in a funciton. There are multiple dataypes in javascript such as string, int, double, and more. Strings can be declared using a single or double quote in javascript.

Arithmetic and Operators

Arithmetic Operators

  • =: this operator assigns a value to a variable. Ex. let x = 10;
  • +: adds numbers. Ex. let z = 5 + 10;
  • *: multiplies numbers. Ex. let z = 5 * 10;
  • /: divides numbers. Ex. let z = 5 / 10;
  • %: modulus (gives the remainder). Ex. let z = 10 % 7; (z would equal 3)

Comparison operators

  • ==: equal to
  • ===: equal to with the same type
  • !=: not equal to
  • >: greater than
  • <: less than
  • If you put an equals sign after > or < you will get greater than or equal to or less than or equal to

Arithmetic

When doing arithmetic math is done in the same order as it normally would. For example if you had 1 + 5 * 2 it would be 11 becuase multiplication comes first.
