HTML Tutorial and CSS Basics

HTML documents consist of tags that create the structure of the page. Most of the tags must have a matching </> tag to end it. HTML elements usually consist of attribute value pairs inside the tag, and content between the 2 tags.

Basic Document Setup

All HTML documents start with the <!DOCTYPE html> tag. This declares that you are writing with html. The next tag after that is the <html> tag. This tag has a closing tag that is the last line of the HTML document. A page is set up in the order of head, body, header, main, footer tags with header, main, and footer contained in the body tag. General basic page layout is pictured above.

Basic Tags

  • <head> This is where metadata on the website goes such as title, or links to stylesheets. The user does not see this
  • <body> This is where all of the viewable information on the page goes. All of the information shown on the page is stored within the body open and closing tag.
  • <h1> This tag is a main heading tag and is supposed to be used once per page. There are 6 types of headings total and they can be used by replacing the 1 in this tag with 2-6. Those are subheadings and are used for structural reasons and are titles or subtitles that you want to display on a page.
  • <p> This is a paragraph tag and this is the type of tag that you would put a body of text in
  • <a> This tag is an anchor tag and is used for links
  • <script> This tag allows a user to embed a javascript script into a page. They can either put code between the 2 tags or link it to a js file.
  • <title> This tag allows the user to add a title to the page. This is shown in the top tab bar.

CSS stands for cascading style sheets and it is a basic way to style the HTML web pages. CSS helps a developer create a unique page that is accessible by all users. It also allows for easy changes to be made to the website globally if set up in a well put together external style sheet.

There are a few ways to get css on a web page. The first way is to use an external style sheet. This is a separate file that has the CSS code that gets linked to the HTML file using a link tag with a rel of stylesheet and href pointing to the file. The second time is putting the CSS between 2 style tags in the head. The last way is to put the css within a style value within the individual tag that you want to customize.

CSS works by targeting elements and changing properties of them. There are 3 ways to target elements which are: by tag, by class or by ID. to target a tag in css you just use the tag name in the CSS modifider. To target a class you put a dot (.) before the class name and then put the properties down. To target an ID you put a # before the ID name and then put the properties. To put the properties down you put 2 curly braces and then put the property name you would like to change and then a colon then the property style then a semicolon. for example, body{background-color: white;}
